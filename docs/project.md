# Project: RoadWolf

RoadWolf is a modern coach booking application built with React, Vite, and TypeScript. It allows users to search for bus routes, select seats, and book tickets.

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** 
  - Tailwind CSS (v4)
  - Emotion (Styled Components)
  - Shadcn UI / Radix UI (Primitives)
  - Material UI (Icons/Components)
- **Routing:** React Router DOM
- **Icons:** Lucide React, MUI Icons

## Key Features

- **Home Page:** Search interface for bus routes.
- **Booking Flow:**
  - Route Selection
  - Seat Selection (`SeatSelectionPage`)
  - Passenger Details (`BookingPage` steps)
  - Ticket Confirmation (`CheckTicketPage`)
- **UI Components:** Comprehensive set of accessible components based on Radix UI and Shadcn UI (in `src/app/components/ui`).

## Project Status

The project has reached **v0.2.0 (Polished MVP)**. It features a fully responsive booking journey with mobile-first verification flows (`src/app`) and specific design implementations derived from Figma.
